# Arena Rubra – C2e-4g1 APK-M4b Mobile UI Microfix

Base: `C2e-4g1 APK-M4 Hard Mobile UI Rebuild`.

## Modifiche
- Il pannello **Opzioni** diventa un overlay alto con scroll interno reale, così su smartphone sono raggiungibili fazioni, comandanti, bot/umano, preset, statistiche e pulsanti.
- Il pannello **Mano** ha priorità visiva/z-index superiore quando aperto.
- Quando il giocatore preme **Gioca carta / Costruisci carta / Gioca ora** da una carta della mano, il pannello Mano si chiude automaticamente e torna disponibile la mappa per scegliere la cella o il bersaglio.

## Non modificato
- Nessuna modifica a gameplay, AI, deck, tattiche, costi, HP/DEF/ATT, mappa logica o regola danno no-overflow.
