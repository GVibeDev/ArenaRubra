# C2d-1b – Title Bar / Game Options UI Polish

Base: C2d-1 validata.

## Obiettivo
Rifinire la UI superiore senza modificare gameplay, deck, AI, economia, tattiche o combattimento.

## Modifiche
- Barra titolo ripulita: titolo centrato `ARENA RUBRA - Starter Game (C2d-1b)`.
- Eliminata la frase descrittiva sotto il titolo.
- `Nuova partita` spostato sulla sinistra della barra titolo.
- `Fine turno` spostato sulla destra della barra titolo.
- Rimosse le copie dei due pulsanti dal pannello opzioni.
- Pannello `Opzioni di gioco` convertito in pannello a scomparsa, aperto di default.

## Gameplay
Nessuna modifica.
