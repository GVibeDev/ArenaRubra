# C2-FINAL-B4 – Agathoi Numerical Balance Pass

Patch di bilanciamento numerico Agathoi applicata sopra C2-FINAL-B3.

## Perimetro

- Nessun nuovo effetto.
- Nessuna nuova meccanica.
- Nessuna modifica AI.
- Nessuna modifica a deck, mano, cap mano, recupero deck o income base.
- Demagos Verde invariato in attesa del futuro pass comandanti.

## Unità modificate

- Carro Falange: ATT 2 → 1.
- Karyon Mobile / Traslazione Karyon: costo abilità 0 → 1 ENE.
- Tholos Sacro: costo 2 → 3.
- Peribolos Murario: costo 2 → 3.
- Cisterna Radicata: costo 3 → 4.
- Selva Simbiotica: costo 5 → 6.

## Tattiche modificate

- AGTAC04 – Bastione Ligneo: costo 3 → 4.
- AGTAC06 – Fortezza Verde: costo 4 → 5.
- AGTAC08 – Carovane di Mercanti: costo 3 → 4.

## Obiettivo

Gli Agathoi conservano la loro identità difensiva e strutturale, ma gli engine permanenti e le fortificazioni non entrano più a costo troppo basso nel nuovo ritmo con income base 3 ENE.
