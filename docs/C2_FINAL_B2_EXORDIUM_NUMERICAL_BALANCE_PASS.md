# C2-FINAL-B2 – Exordium Numerical Balance Pass

## Obiettivo

Riallineare Exordium al nuovo ritmo dello Starter Game con income base 3 ENE, senza introdurre nuove meccaniche e senza alterare l’identità militare/offensiva della fazione.

## Principio

Exordium deve continuare a colpire forte, sfondare DEF e premiare pressione diretta, ma gli effetti offensivi e di shred non devono generare valore gratuito su corpi già efficienti.

## Comandanti

Il Pretore di Aurex resta invariato. Il bilanciamento dei comandanti è rimandato a un pass dedicato, in cui verranno valutate abilità attive e un comandante aggiuntivo per fazione.

## Modifiche unità

- Cursor: ATT 2 → 1.
- Legionario Pesante / Colpo Pesante: costo abilità 0 implicito → 1 ENE.
- Legionario d’Assedio / Colpo d’Assedio: costo abilità 2 → 1 ENE.
- Carro Medio Exordium / Sfondamento: costo abilità 0 implicito → 1 ENE.
- Testudo / Frattura della Linea: costo abilità 0 → 1 ENE.
- Artiglieria Exordium / Bombardamento: costo abilità 0 implicito → 2 ENE.
- Avamposto Exordium: costo 3 → 4.
- Orgoglio di Aurex: costo 5 → 6.

## Modifiche tattiche

- EXTAC04 – Obliteratore: costo 4 → 5.
- EXTAC08 – Rifornimenti in arrivo: costo 3 → 4.
- EXTAC10 – Campo statico: costo 2 → 3.

## Non modificato

- Nessuna nuova meccanica.
- Nessun nuovo effetto.
- Nessuna modifica al Pretore di Aurex.
- Nessuna modifica alle altre fazioni.
- Nessuna modifica a UI, AI, deck, mano, income base, cap mano o recupero deck.
