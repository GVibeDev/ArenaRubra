# F9J1 – Card Pool Screen Foundation

Base: `C2-STABLE-1-F9I2a-APK-M4c`

## Obiettivo
Aggiungere una schermata `Pool carte` read-only per consultare l'intero catalogo carte con preview renderer e riferimenti asset.

## Funzioni
- Browser di tutte le carte del catalogo.
- Filtri per fazione, tipo carta e ruolo.
- Ricerca testuale su nome, ID, categoria, effetto e note.
- Preview renderer della carta selezionata.
- Visualizzazione path frame/art/fallback e formato asset consigliato.
- Copia JSON della carta selezionata.
- Copia manifest asset carte.

## Non modificato
Gameplay, AI, deck rules, bilanciamento, setup partita, storage persistente e runtime carte.
