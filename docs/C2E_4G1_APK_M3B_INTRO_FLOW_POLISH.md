# Arena Rubra – C2e-4g1 APK-M3b Intro Flow Polish

Base: C2e-4g1 APK-M3 Intro / Splash / Music.

## Modifiche
- Musica 432 Hertz Rift impostata con preload, autoplay, loop e tentativo JS immediato all'avvio.
- Fallback invisibile: se la WebView/browser blocca l'autoplay, il primo touch/key prova a sbloccare l'audio senza mostrare pulsanti o messaggi.
- Splash G vibe dev senza pulsanti: durata 3 secondi, qualsiasi click/touch passa alla title.
- Transizione dev -> nero -> title con fade; la title ha fade più lento e zoom cinematico più lungo.
- La schermata Arena Rubra resta sullo sfondo finché non viene premuto il solo pulsante "Starter Game : Blueprint".
- Rimossi pulsanti musica ON/OFF, salta intro, avvia intro e testi dinamici/pronto.
- Footer discreto "G Vibe Dev" in basso.

## Non modificato
Gameplay, AI, deck, tattiche, bilanciamento, mappa logica e regola danno no-overflow.
