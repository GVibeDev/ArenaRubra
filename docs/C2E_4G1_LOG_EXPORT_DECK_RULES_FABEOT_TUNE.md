# Arena Rubra – C2e-4g1 Log Export / Deck Rules / Fabeot Tune

## Base
C2e-4g AI Integration / Regression Pass.

## Perimetro
Patch mirata per log export, regole di generazione deck e due aggiustamenti Fabeot richiesti.

## Modifiche
- UI log: aggiunti pulsanti **Copia log** ed **Esporta log .txt** nel pannello Comandi e log.
- Export log TXT: include header partita con build, data export, match id, round, fazioni, comandanti, modalità, preset, vincitore, tipo vittoria, PS, pressione, ENE e unità finali.
- Deck: lo starter loadout selezionato per fazione non viene più inserito nel deck perché disponibile via pulsanti starter.
- Deck copy rules: comandanti, pivot ed elite massimo 1 copia; tutte le altre carte, tattiche comprese, massimo 2 copie.
- Fabeot: Adepto Fabeot costo 1 ENE.
- Fabeot: Strozzatura Logistica resta tattica default costo 1 ENE; il bersaglio marchiato, se attacca, fa guadagnare +2 ENE al Fabeot proprietario del marchio.

## Non modificato
- Nessuna modifica alla regola danno normale no-overflow DEF → HP.
- Nessuna modifica generale a HP, DEF, ATT.
- Nessun nerf/buff numerico ad altre fazioni.
- Nessuna modifica alla mappa.
- Nessun refactor AI strutturale oltre la base C2e-4g.
